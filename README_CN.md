# **osu\!mania 回放数据分析工具 (Replay Data Analyzer)**

本项目是一个用于解析并分析 osu\!mania 游戏回放文件（.osr）的 Python 脚本。该工具能够提取回放中的按键实时数据，计算每个按键的持续时间（Duration），并生成高分辨率的统计分布折线图，旨在帮助 VSRG（Vertical Scrolling Rhythm Game）玩家分析其按键习惯与准度表现。~~100%由gemini氛围编程~~

## **功能简介**

* **回放解析**：自动解析 .osr 文件的二进制头部信息及 LZMA 压缩的动作序列数据。  
* **数据统计**：基于状态机逻辑精确计算按键按下至弹起的持续时间（支持多键位并行处理）。  
* **可视化渲染**：利用 Matplotlib 生成按键持续时间分布图，支持 1ms 级别的精细刻度显示。  
* **高分辨率输出**：预设从 HD (720P) 到 4K (2160P) 的多种渲染分辨率选项，确保图像质量。

## **环境要求**

运行本项目需要 Python 3.x 环境及以下第三方库：

* matplotlib：用于图表绘制与渲染。  
* 内置模块：struct, lzma, os, sys, collections 等。

**安装依赖：**

pip install matplotlib

## **使用说明**

1. **准备回放文件**：请确保已拥有 .osr 格式的回放文件。  
2. **启动脚本**：  
   python osu\_replay\_parser.py

3. **输入路径**：根据提示，将 .osr 文件拖入终端或手动输入完整路径。  
4. **选择分辨率**：根据需要选择 1-4 档输出分辨率（默认为 1080P）。  
5. **查看结果**：脚本将在同级目录下生成名为 \[文件名\]\_analysis.png 的分析图表。

## **优化与改进**

本项目灵感来源于 [adgjl7777777/VSRG\_Total\_Analyzer](https://github.com/adgjl7777777/VSRG_Total_Analyzer)，并在其基础上进行了以下优化：

1. **性能与兼容性**：改进了 ULEB128 与变长字符串的读取逻辑，增强了对不同版本回放格式的兼容性。  
2. **视觉表现力**：  
   * 引入了主次刻度定位器（MultipleLocator），使 X 轴（时间间隔）和 Y 轴（频次）的观察更加直观。  
   * 优化了绘图参数，包括线宽随分辨率动态缩放、图例位置自适应、以及网格线透明度调整。  
3. **交互体验**：简化了命令行交互流程，增加了路径自动清理功能（去除路径引号等），并支持多分辨率导出。  
4. **代码规范**：采用了更严谨的模块化结构，并提供了详尽的中文专业注释，方便开发者进行二次开发。
